A Group Where DDH Assumption Does (NOT) Hold

2020-07-05 · 7 min read
Cryptography

Note that DDH Assumption is a “belief” and not a “fact.” If it is proven that such groups exist where DDH assumption holds, then this proof will also imply that P≠NP\mathcal{P} \ne \mathcal{NP}P​=NP. So we can only find that the DDH hypothesis does NOT hold in some groups, and it seems impossible to prove that the DDH hypothesis holds in some groups.

We already know that DDH assumption does not hold in groups where computing DLOG is easy, so does in the groups have parings.

We present here that in a common group Zp∗\mathbb{Z}_p^*Zp∗​ (where ppp is a prime), DDH assumption does not hold as well.
